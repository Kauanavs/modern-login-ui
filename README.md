# ☕ Modern Login UI – Coffee Theme

Interface de login moderna com design minimalista e foco em experiência do usuário.

O layout combina uma estética clean com uma imagem lateral que traz identidade visual e equilíbrio ao design.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3

---

## 🎯 Objetivo do projeto

Criar uma interface de login visualmente agradável, com foco em:

- Simplicidade
- Usabilidade
- Design moderno

---

## ✨ Funcionalidades

- Layout dividido (formulário + imagem)
- Campos estilizados com ícones
- Botão de ação destacado
- Link de recuperação de senha
- Design responsivo

---

## 🎨 Destaques de UI/UX

- Paleta de cores neutra e elegante
- Tipografia limpa
- Uso de imagem para enriquecer o layout
- Estrutura organizada e intuitiva
